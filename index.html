Proyectos Personales

Además de este sitio web, también estoy trabajando en otros proyectos. Aquí hay algunos de ellos:

LA CAFFETTIERA DE RUMEN

La Caffettiera de Rumen es una aplicación web responsive creada con Django que simula la página de un café moderno. Incluye una sección de productos, formulario de contacto funcional y panel de administración. El proyecto demuestra el desarrollo completo con backend en Python/Django y su despliegue en producción.

Repositorio GitHub | Acceso Web

CERTIFICADO DIGITAL

Explicación guiada paso a paso sobre cómo obtener y configurar un certificado digital y realizar una firma electrónica en España, utilizando un avatar animado como narrador principal. Este vídeo forma parte de un trabajo colaborativo realizado por Stanislav Muntyan, Arantxa Quijada, Javier Canudo y Rumen Orachev.

Repositorio GitHub | Youtube

RED EMPRESARIAL

Diseño completo de una red informática simulada para una empresa de 75 personas, utilizando Cisco Packet Tracer y narrado mediante un avatar animado. Trabajo colaborativo realizado por Stanislav Muntyan, Arantxa Quijada, Javier Canudo y Rumen Orachev.
